⚠️ This panel is HTTP by default. If you try HTTPS you’ll get ERR_SSL_PROTOCOL_ERROR. Use http:// unless you set up a reverse proxy + SSL.
Change port
Example: 2323
1) Edit .env and set:
PORT=2323
2) Restart the panel (systemd or node). If you run it manually:
cd ~/sootypage-game-panel
node src/index.js
If you made a systemd service for the panel, restart it:
sudo systemctl restart sootypage-game-panel
